# Documentation Index

This folder contains project documentation and delivery artifacts.

## Core Guides

- `QUICK_START.md`: fast run instructions for simulator, video processor, and Pi runtime.
- `DEBUG_GUIDE.md`: deep debugging and visualization workflow.
- `VERIFICATION.md`: validation checklist and expected behavior.

## Delivery Artifacts

- `CHANGES_SUMMARY.md`: detailed change log of major updates.
- `FILE_CHANGES.md`: file-level modification summary.
- `CODE_REVIEW.md`: high-level review notes.
- `COMPLETION_SUMMARY.txt`: completion/report text artifact.

For architecture overview and versioning, start from the repository root `README.md`.
